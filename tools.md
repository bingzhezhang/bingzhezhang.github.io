## 🛠 Open-source Tools

### 🔧 Tool 1: Seismic Isolation Bearing Parameter Identification GUI
This project provides a GUI tool for constitutive parameter identification and visualization of seismic isolation bearings.
- GitHub: [https://github.com/bingzhezhang/Bearing-Constitutive-Parameter-Identification-System](https://github.com/bingzhezhang/Bearing-Constitutive-Parameter-Identification-System)

### 🔧 Tool 2: APDL Models for Laminated Rubber Bearings
This project provides ANSYS APDL scripts for nonlinear analysis of bonded and unbonded laminated rubber bearings.
* GitHub: [https://github.com/bingzhezhang/ANSYS-APDL-Parametric-Modeling-and-Analysis-of-Laminated-Rubber-Bearing](https://github.com/bingzhezhang/ANSYS-APDL-Parametric-Modeling-and-Analysis-of-Laminated-Rubber-Bearing)
